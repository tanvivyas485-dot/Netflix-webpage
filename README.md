# Netflix-webpage
 This is Netflix-style homepage built with HTML, CSS, and vanilla JS. It features a dark theme, a featured banner, and scrollable movie rows with hover effects. I focused on making it responsive and interactive, adding a drag-to-scroll feature for the movie lists. 

# 🍔 Netflix Clone - Student Project

Hey! This is my attempt at building a Netflix-like homepage from scratch. I wanted to practice my HTML, CSS, and JavaScript skills by recreating a familiar interface.

## ✨ Features

- **Dark Theme UI** - Classic Netflix look and feel.
- **Responsive Navigation** - Sticky navbar that works on mobile and desktop.
- **Featured Banner** - Hero section with a gradient overlay and play button.
- **Scrollable Movie Rows** - Horizontal scrolling lists for different categories.
- **Interactive Elements** - Hover effects on movie cards and drag-to-scroll functionality.
- **Single File** - Everything is in one `index.html` file for simplicity.
